          About×

          What?

          This site visualises the last.fm tags for bands that are playing SWN Festival in 2012. The idea is that you can see how bands have been tagged and then see which other artists share those tags. Not all the bands playing have tags on Last.FM, so can't be selected and won't show up when you choose a tag.

          Why?

          Created as an attempt to help make the decisions on which band to see at SWN easier.

          How?

          The tag clouds use Jason Davies's excellent D3 word cloud plugin. Most of the .js for creating the clouds was modified from his example. The rest of the code was poorly hacked together by me, it's basically just bootstrap for styling and a sprinkling of jquery for the band list interaction. If I had time, I'd improve it, but I don't so probably won't.

          When?

          Put together over a couple of hours on the 22nd and 23rd September 2012 while I was avoiding other work
